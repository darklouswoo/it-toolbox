# Changelog - v2.1.0

**发布日期**: 2026-03-24

IT Toolbox v2.1.0 从 v2.0.0 的 96 个工具扩展到 106 款工具，新增 10 个实用工具。

---

## 新增工具（10个）

### 文件完整性校验
- 上传文件计算 MD5/SHA-256 哈希值
- 支持批量文件上传
- 输入预期哈希值进行校验对比
- 实时显示校验结果（匹配/不匹配）
- 使用 Web Crypto API 前端计算

### JWT 签名验证
- 支持 HMAC (HS256/HS384/HS512) 签名验证
- 支持 RSA (RS256/RS384/RS512) 签名验证
- 解析并展示 Header 和 Payload
- 检测 Token 过期状态
- 显示签发时间和过期时间

### 密码强度分析
- 使用 zxcvbn 算法分析密码强度
- 实时估算多种攻击场景下的破解时间
- 字符组成分析（大小写/数字/符号）
- 提供改进建议
- 检测常见密码模式

### 随机 JSON 生成
- 可视化定义 Schema 结构
- 支持多种数据类型（字符串/数字/布尔/日期/邮箱/姓名/电话/地址/URL/UUID）
- 批量生成多条数据
- 支持下载 JSON 文件
- 一键复制生成结果

### SQL 测试数据生成
- 可视化定义表结构
- 支持多种 SQL 数据类型
- 生成 INSERT 语句
- 支持单条/批量 INSERT 模式
- 可配置生成行数
- 支持下载 SQL 文件

### 正则从样本生成
- 输入多个样本字符串
- 自动推断匹配的正则表达式
- 实时测试正则匹配结果
- 支持手动编辑正则
- 一键复制生成的正则

### 数字单位换算
- 支持 8 大类单位换算
  - 长度：千米/米/厘米/毫米/英里/码/英尺/英寸/海里
  - 重量：吨/千克/克/毫克/磅/盎司/斤/两
  - 温度：摄氏度/华氏度/开尔文
  - 速度：米每秒/千米每时/英里每时/节/马赫
  - 面积：平方千米/平方米/公顷/英亩/亩
  - 体积：立方米/升/毫升/加仑
  - 时间：年/月/周/天/小时/分钟/秒/毫秒
  - 压力：帕斯卡/千帕/兆帕/巴/psi/大气压
- 实时显示全部单位换算结果
- 支持单位交换

### 数据存储换算
- 支持 Bit/Byte/KB/MB/GB/TB/PB/EB 换算
- SI 单位制（1000 进制）与二进制单位（1024 进制）切换
- 自动选择最佳显示单位
- 显示精确字节数
- 一键复制换算结果

### 色彩空间换算
- 支持 RGB/HSL/HSV/HEX/CMYK 全量互转
- 实时颜色预览
- 滑块可视化调节
- 支持颜色选择器
- 一键复制各种格式

### 多格式时间转换
- 支持 10+ 种时间格式互转
  - ISO 8601
  - RFC 2822
  - HTTP Date
  - SQL DateTime/Timestamp
  - Unix Timestamp（秒/毫秒）
  - Excel Serial
  - Julian Date
  - 本地时间
- 支持自定义格式（YYYY-MM-DD HH:mm:ss 等）
- 自动解析多种输入格式
- 一键复制各种格式

## 技术亮点

### 纯前端实现
本次新增的 10 个工具全部采用纯前端实现，无需后端 API 支持：
- 文件哈希计算使用 Web Crypto API
- JWT 验证使用 SubtleCrypto API
- 所有计算在浏览器端完成，保护用户隐私

### 性能优化
- 大文件哈希计算采用流式处理
- 单位换算结果实时缓存
- 组件懒加载减少首屏体积

### 用户体验
- 统一的交互设计语言
- 一键复制功能
- 支持键盘快捷操作
- 响应式布局适配移动端
